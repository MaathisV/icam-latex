# Icam Latex

This repository contains **unofficial** LaTeX templates for the [Institut Catholique des Arts et Métiers (Icam)](https://www.icam.fr).

I'm not the original author of theses. My work is heavily inspired from [qaniere/utc-latex](https://github.com/qaniere/utc-latex), while trying to be as close as possible to [Icam's graphical guidelines](https://brandcenter.icam.fr/document/charte-graphique/)


Feel free to open issues and of course, pull requests are very welcome.
